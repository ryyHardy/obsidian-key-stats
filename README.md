# Typing Stats

This is an Obsidian plugin that calculates advanced typing stats as you take notes in your vault.
So, you can improve your typing and be productive at the same time.

It does not work like typing websites such as keybr or Monkeytype, because there is no "target text" to compare to. The analysis is just done on the fly.

- Instead of directly reading keystrokes, the plugin examines changes in the actual content of the note. We only care what actually gets put in the file.
- Instead of using a dictionary of pre-defined words, the plugin infers errors by detecting things like backwards cursor movement and deletions. This way, we are not tied to specific languages or vocabularies. Only what the user considers an error.
  - You _could_ cheat your stats up by just typing a bunch of gibberish, but that doesn't really benefit your improvement. There is no leaderboard so it only hurts you.
